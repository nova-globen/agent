## Reading Guide

1. Start with `ubiquitous-language.md` for the canonical vocabulary.
2. For each domain concept, the authoritative model is in `contexts/<NN>-<name>/`.
3. Domain events are named `NounPastTenseEvent` (e.g. `GreetingRequested`, `UserRegistered`).
4. Implementation terms (framework/library names) must not appear in the domain model.

## Conventions

- Domain event names: `NounPastTenseEvent`
- Aggregate roots: `<AggregateName>` (no suffix)
- Value objects: descriptive noun (e.g. `EmailAddress`, `Money`)
- Commands: `VerbNoun` (e.g. `RegisterUser`, `PlaceOrder`)
- Queries: `GetNoun` / `ListNouns` (e.g. `GetUser`, `ListOrders`)
