Prove the autopilot loop works end-to-end: implement a trivial .NET feature, verify it, commit it, and
write a handoff prompt — one complete session. This feature exists solely to validate the kit machinery;
replace it with your first real feature when adopting.

## Scope

- `src/Platform.Hello/Greeter.cs` — a `Greeter` class with a single method `Greet(string name)` that

- `tests/Platform.Hello.Tests/GreeterTests.cs` — one xUnit `[Fact]` asserting `Greet("World")` returns

- Any persistence, HTTP endpoints, or UI.
